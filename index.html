To People Considering Career Change to Developers

I am taking a full stack web developers class at a bootcamp in Atlanta. Before I decided to do this, I was an attorney licensed in New York, New Jersey. I am still licensed, but I do not think I am going to practice law any more. I did not know anything about programming before I took this class. I was like a baby in front of a computer.

I am still a beginner, and there are not so many things I can do, most of the time I am like this.

But sometimes, when I fixed an error, I am like this.

What should we learn as a beginner?

JavaScript. It is the only lanuguage that runs in browsers. As a web developer, JavaScript is a basic, and a must.

What programms do I need to begin programming?

Text Editor: To write source code, you need a text editor. There are many good text editors you can choose, but I use Visual Studio Code. You can download VSC here.

With the Visual Studio Code, you can write a code and push it to github.
